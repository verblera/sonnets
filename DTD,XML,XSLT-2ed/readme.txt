This folder contains:
- seven xml files
- one DTD files
- one XSLT file

DTD defines and check structure of XML files
XSLT file defines how XML should be presented 
each XML is marked up.

This XSLT file gives a short description of a particular
type of sonnet and source of information;
it also creates a metric scheme for a sonnet 
(types of stanzas, scheme of rhymes);
it counts a number of lines.
